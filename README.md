# DepthAI Data

This repository holds various data and firmware files that are used by [depthai-core](https://github.com/luxonis/depthai-core).

## Update

To update the data in this repository run the script with path to the `cmake` directory inside `depthai-core`:

```
./extract_cmake_urls.sh -c ~/luxonis/depthai-core/cmake update
```
