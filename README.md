# Depthai data

This repository holds various data and firmware files that are used by [depthai-core](https://github.com/luxonis/depthai-core).

The files were extracted from the CMake from commit [`465b6a1` (v3.2.1)](https://github.com/luxonis/depthai-core/commit/465b6a1ce70a8d99ba06849f2b6b0ffe7603e1ea).

## Update

To update the binaries here just run the script with path to the `cmake` dir inside the `depthai-core`:

```
./extract_cmake_urls.sh -c ~/luxonis/depthai-core/cmake --output-dir .  update

```
